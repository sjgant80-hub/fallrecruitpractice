## For the end user

Run your recruitment agency finances from one HTML file. 9 tabs cover placements, perm fees, temp margins, invoicing, consultants, expenses, and agency P&L.

### Key features

| Tab | What it does |
|---|---|
| Dashboard | KPI overview — perm fees, temp margin, outstanding, AWR alerts |
| Placements | Active/completed placements with AWR week tracking |
| Perm Fees | Permanent placement fees with salary, %, rebate status |
| Temp Margin | Weekly charge rate vs pay rate with margin calculation |
| Invoices | Invoice tracking with sent/paid/overdue status |
| Consultants | Desk assignment, perm/temp revenue, commission % |
| Expenses | Agency overheads by category |
| Agency P&L | Rolling 12-month: perm fees + temp margin − expenses = net |

### Key rules

- **Conduct Regulations 2003**: fee terms in writing, transfer fees (Reg 10)
- **AWR 2010**: 12-week tracking, margin impact on equal pay
- **EAA 1973 s.6**: no charging work-seekers
- **Late Payment Act 1998**: interest on overdue invoices
- **HMRC**: PAYE/RTI for temp payroll, VAT on recruitment services
